# Simple Lockscreen Tweak

A tiny, lightweight Objective-C tweak for iOS lockscreen. 

## 📖 Description
This tweak simply moves the lockscreen time and date to the right side of the screen. It uses minimal hooks, making it extremely lightweight and battery-friendly. There are no settings or configuration options—it works right out of the box.

## 📱 Compatibility
* **Target Environment**: Specifically built for **iOS 16 Rootless (Dopamine)**.
* **Tested Devices**:
  * iOS 16.1.1 (A15 Bionic)
  * iOS 16.3 (A11 Bionic)

⚠️ **Warning**: This is my very first tweak! Please do not try this on non-iOS 16 rootless environments unless you know what you are doing.

## 🛠️ Build Environment
If you want to compile it yourself, here is the environment I used:
* **OS**: Windows WSL (Ubuntu 24.04 LTS)
* **Framework**: Theos
* **SDK**: iOS 16.5 SDK
